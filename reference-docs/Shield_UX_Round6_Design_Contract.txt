Create a modern enterprise government-grade web platform that blends Microsoft Fluent design principles, USWDS accessibility standards, and modern operational dashboard UX patterns commonly seen in Azure Portal, Microsoft Defender XDR, and Datadog.

The platform must feel:

professional
trustworthy
mission-focused
operationally mature
accessible
scalable
data-driven
calm and uncluttered

Avoid flashy startup aesthetics, cyberpunk styling, excessive animations, neon colors, gaming UI patterns, or overly artistic layouts.

The design should support three distinct user experiences while maintaining one cohesive design language:

Public / External Experience
Operational Dashboard Experience
Executive Leadership Experience
